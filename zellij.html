zellij cheatsheet

Keybindings and commands for the Rust terminal multiplexer

Homezellij.dev Installbrew install zellij
Modes: zellij is modal. Enter a mode with Ctrl+letter, then press the next key. Esc or Enter returns to Normal mode.

Mode switches (from Normal)

Ctrl+p
Pane mode
Ctrl+t
Tab mode
Ctrl+n
Resize mode
Ctrl+h
Move mode
Ctrl+s
Scroll / search mode
Ctrl+o
Session mode
Ctrl+g
Lock / unlock input
Ctrl+q
Quit zellij

Scroll / search mode

j / k
Scroll down / up
Ctrl+f / Ctrl+b
Page down / up
d / u
Half page down / up
g / G
Top / bottom
s
Search in buffer
e
Open pane buffer in editor

Copy / paste

Mouse drag
Select text → auto-copied (copy_on_select true)
Shift-drag
Bypass zellij, native terminal select
Cmd+V / Ctrl+Shift+V
Paste (terminal handles it)
Ctrl+s then s
Enter search in scrollback
n / p
Next / prev match (in search)
c / w
Toggle case / whole-word
Ctrl+s then e
Open scrollback in $EDITOR — yank there
Ctrl+g
Lock input — pass paste shortcuts through to vim/tmux

Config keys in config.kdl: copy_command "pbcopy" (mac) / "wl-copy" / "xclip -selection clipboard", copy_clipboard "system", copy_on_select true.

CLI

zellij zellij -s mywork zellij attach mywork zellij attach -c mywork zellij ls zellij kill-session mywork zellij kill-all-sessions zellij delete-session mywork zellij -l compact zellij action new-tab

Config & layouts

zellij setup --dump-config > \ ~/.config/zellij/config.kdl zellij setup --dump-layout default ~/.config/zellij/layouts/*.kdl zellij -l ~/my.kdl # built-in layouts: zellij -l default zellij -l compact zellij -l disable-status-bar
Tip: Esc always returns to Normal mode. Stuck? Ctrl+g toggles input lock so passthrough apps (vim, tmux) regain control.
