tmux cheatsheet

Keyboard shortcuts and commands for the terminal multiplexer

Prefix key: Ctrl+b by default. Shown as pre below. Press prefix, release, then press the next key.

Sessions

pre d
Detach from session
pre s
List / switch sessions
pre $
Rename current session
pre (
Previous session
pre )
Next session

Session CLI

tmux new -s mywork tmux ls tmux attach -t mywork tmux attach tmux kill-session -t mywork tmux kill-server

Windows

pre c
Create new window
pre ,
Rename current window
pre n
Next window
pre p
Previous window
pre 0–9
Jump to window by number
pre w
Interactive window list
pre &
Kill current window
pre l
Last active window

Panes — splitting

pre %
Split vertically (left / right)
pre "
Split horizontally (top / bottom)
pre x
Kill current pane
pre z
Toggle pane zoom (fullscreen)
pre !
Break pane into its own window
pre q
Show pane numbers

Panes — navigation

pre ← ↑ ↓ →
Move between panes by direction
pre o
Cycle to next pane
pre ;
Toggle last active pane
pre {
Swap pane with previous
pre }
Swap pane with next
pre Space
Cycle through layouts

Panes — resizing

pre Ctrl+←
Resize left (hold to repeat)
pre Ctrl+→
Resize right
pre Ctrl+↑
Resize up
pre Ctrl+↓
Resize down
:resize-pane -D 10
Resize down by 10 (command mode)

Copy mode & scrolling

pre [
Enter copy mode (scroll / select)
q
Exit copy mode
Space
Start selection (in copy mode)
Enter
Copy selection
pre ]
Paste most recent buffer
/ or ?
Search forward / backward
g / G
Jump to top / bottom

Misc

pre :
Enter command mode
pre ?
List all key bindings
pre t
Show clock
pre r
Reload config (if bound)
:source ~/.tmux.conf
Reload config manually

~/.tmux.conf starter

# Remap prefix to Ctrl-a unbind C-b set -g prefix C-a bind C-a send-prefix # Mouse support set -g mouse on # Start windows at 1, not 0 set -g base-index 1 setw -g pane-base-index 1 # Reload with prefix + r bind r source-file ~/.tmux.conf # Vim-style pane navigation bind h select-pane -L bind j select-pane -D bind k select-pane -U bind l select-pane -R
Tip: pre ? inside tmux lists every binding live. q exits the list.
