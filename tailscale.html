tailscale cheatsheet

CLI for the WireGuard-based mesh VPN

Hometailscale.com Installtailscale.com/download Adminlogin.tailscale.com/admin
Mental model: up / down control the connection. set persists flags without bouncing the connection. Admin console at login.tailscale.com/admin for ACLs, keys, DNS.

Connection

tailscale up tailscale up --authkey tskey-... tailscale up --hostname=laptop tailscale up --ssh tailscale down tailscale login tailscale logout

Status & info

tailscale status tailscale status --json tailscale status --peers=false tailscale ip tailscale ip -4 host tailscale whois 100.64.0.1 tailscale netcheck tailscale ping host tailscale version

Persistent config (set)

tailscale set --hostname=laptop tailscale set --ssh tailscale set --accept-dns=true tailscale set --accept-routes tailscale set --shields-up tailscale set --operator=$USER

Exit nodes

# advertise this node as exit tailscale set --advertise-exit-node # use an exit node (must be approved) tailscale set --exit-node=home-pi # allow LAN while using exit node tailscale set \ --exit-node-allow-lan-access # stop using exit node tailscale set --exit-node=

Subnet routing

# advertise routes from this node tailscale set \ --advertise-routes=10.0.0.0/24 # enable IP forwarding on Linux echo 'net.ipv4.ip_forward = 1' | \ sudo tee -a /etc/sysctl.d/99-ts.conf sudo sysctl -p /etc/sysctl.d/99-ts.conf # accept advertised routes tailscale set --accept-routes

SSH

# enable SSH server on this node tailscale set --ssh # connect (auth via tailnet ACLs) tailscale ssh user@host ssh user@host # works once enabled

Serve (tailnet-only)

tailscale serve 3000 tailscale serve --bg 3000 tailscale serve --https=443 \ /api http://localhost:8080 tailscale serve status tailscale serve reset

Funnel (public internet)

# expose to public via *.ts.net tailscale funnel 443 on tailscale funnel --bg 3000 tailscale funnel status tailscale funnel reset

File sharing & drive

# push a file to a peer tailscale file cp notes.md host: # receive files sent to this node tailscale file get . # tailscale drive (SMB-like) tailscale drive share docs ~/Documents tailscale drive list

Debug / troubleshoot

tailscale netcheck tailscale ping --tsmp host tailscale bugreport tailscale debug daemon-logs journalctl -u tailscaled -f # macOS client logs log stream --predicate \ 'subsystem contains "tailscale"'
